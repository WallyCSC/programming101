Elementos de citação

Estamos aprendendo HTML. Perceba que a palavra HTML está sublinha. Usando hover sobre ela é possível perceber que ele é um hipertexto, com link. Para fazer isso, ver código HTML no VS Code. Podemos modificar também para outras abreviações. Por exemplo: WWW.

Posso usar também a tag address. Veja o exemplo:

Rua do Exemplo, 123, Cidade, Estado.

Para fazer o endereço acima, utilizei a tag citada.

Outra tag é a tag cite: O filme Titanic com Leonardo DiCaprio. Também existe a tag q, que é usada para fazer citação de fala. Exemplo: Jack disse para Rose: Por que não consigo subir na porta?
Para textos destacados, fazemos o seguinte usando a tag blockquote:

Lorem ipsum dolor sit amet consectetur adipisicing elit.
