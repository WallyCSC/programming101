Tudo Sobre Imagens

Sites são muito visuais, então nada mais importante do que o uso de boas imagens.

Vamos aos fundamentos e tags usados para incluir imagens.

Para se incluir uma imagem, deve-se atentar para a tag específica para ela. Essa tag NÃO possui fechamento, portanto é uma exceção às muitas outras tags que existem em HTML. Ela funciona com menor que + img + src + = +"". Entre as aspas devemos colocar o caminho para chegar a imagem, em outras palavras, de onde o site vai carregar a imagem que vai exibir.

Uma possibilidade é incluir entre as aspas da tag o link da imagem. O site terá que, ao ser carregado, buscar o link e exibir a imagem. Essa não é uma escolha aconselhável, uma vez que o proprietário do link utilizado pode removê-la de la, ou até mesmo o site de onde foi retirada deixe de existir. Assim, uma melhor opção é criar, dentro do projeto, uma pasta chamada imagens, na qual nela serão salvas as imagens do site.

Feito isso, o caminho que colocaremos entre as aspas da tag img será não mais o do link, mas o local na pasta imagens do projeto, assim garantindo que ela não será perdida.

É de suma importância atentar para o fato de que as imagens precisam ter muitas vezes a sua qualidade reduzida. Isso evita demora no carregamento das páginas. Sites que reduzem qualidade de imagem devem ser utilizados. Também é importante lembrar que devem ser usados como fonte das imagens apenas sites que as disponibilizam de forma gratuita e com uso livre, sem direitos autorais.

Mudando o Tamanho da Imagem Dentro da tag

Para fazer isso, após as aspas com o local da imagem, deve-se escrever "width" + a quantidade de pixels, apenas em número, em integer, isso mudará a largura da imagem. Para a aultura faz-se o mesmo utilizando a palavra "height" + a quantidades de pixels.

Existe também o atributo alt="" onde pode-se incluir descrição de imagens para deficientes visuais.

Onde Conseguir Imagens

Sites como o Pexels fornecem imagens gratuitas. Outro site, esse para ícones, é o FlatIcon. É possível ainda incluir gifs, imagens png.
